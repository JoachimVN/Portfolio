Privacy Policy

CHORIDOR  ·  Last updated June 13, 2026

This policy describes what information CHORIDOR ("the game") collects and how it is used.

1. Information We Collect

CHORIDOR does not require accounts or registration. The only data processed during normal use is:

  • Room codes — temporary 3-character codes generated for online sessions. These exist only for the duration of the game and are not stored after the session ends.
  • Game state — board positions and moves, transmitted over WebSocket to sync two players. Not stored after the session ends.

2. Analytics

This site uses Vercel Analytics to collect anonymous usage data, including page views, referrer, browser type, operating system, and approximate country (derived from your IP address — the IP itself is not stored). No cookies are used, and no personally identifiable information is collected. This data is used solely to understand how the site is used.

3. Discord Integration

If you access CHORIDOR through Discord Activities, Discord may share your Discord user ID and username with the application as part of the OAuth authentication flow. This information is used only to identify you within the current game session and is not stored persistently.

Discord's own data practices are governed by Discord's Privacy Policy.

4. Data Sharing

We do not sell, share, or transfer any data to third parties, except as described above (Vercel Analytics, Discord).

5. Data Retention

No personal data is stored by CHORIDOR. All session data (room codes, game state) is held in memory only for the duration of the active game and discarded when the session ends.

6. Children's Privacy

CHORIDOR does not knowingly collect data from children under 13. If you believe a child has provided personal information, contact us and we will take appropriate action.

7. Changes

This policy may be updated at any time. The date at the top of this page reflects the most recent revision.
