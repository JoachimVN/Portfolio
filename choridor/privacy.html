Privacy Policy

CHORIDOR  ·  Last updated June 22, 2026

This policy describes what information CHORIDOR ("the game") collects and how it is used. It applies to the web version at joavn.dev/choridor and the Discord Activity.

1. Information We Collect

CHORIDOR does not require accounts or registration. The data processed during normal use is:

  • Room codes — temporary 3-character codes generated for online sessions. These exist only for the duration of the game and are not stored after the session ends.
  • Display names — the name you enter (or your Discord username in the Discord Activity) is transmitted to the server and shared with the other player(s) in the same room for the duration of the session. It is not stored after the session ends.
  • Game state — board positions and moves, transmitted over WebSocket to sync players and spectators. Not stored after the session ends.

2. Analytics

This site uses Vercel Analytics to collect anonymous usage data, including page views, referrer, browser type, operating system, and approximate country (derived from your IP address — the IP itself is not stored). Vercel Analytics uses no cookies, and no personally identifiable information is collected. This data is used solely to understand how the site is used.

CHORIDOR also uses PostHog (hosted in the European Union) to understand how the game is played and to improve it. Some statistics are sent from the game server (for example, online matches starting and ending). In addition, the game running in your browser sends PostHog anonymous product and gameplay events — for example, page views, which mode you choose (local, online, or vs. AI), when a game starts and ends, how it ends (reaching the goal or surrendering), how long it lasts, the number of moves and walls used, opening the How to Play guide, and copying an invite link. To recognize returning visitors and measure retention, PostHog stores a randomly generated identifier in your browser (using a cookie and local storage). This identifier is not linked to your real-world identity, your IP address is anonymized, and your display name and room code are never attached to these events.

3. Discord Integration

If you access CHORIDOR through the Discord Activity, Discord shares your username and avatar URL with the application as part of the OAuth authentication flow. This information is used only to display your name and avatar to other players in the same session and is not stored persistently after the session ends.

Discord's own data practices are governed by Discord's Privacy Policy.

4. Data Sharing

We do not sell, share, or transfer any data to third parties, except as described above (Vercel Analytics, PostHog, Discord).

5. Data Retention

No personal data is stored by CHORIDOR. All session data (room codes, game state) is held in memory only for the duration of the active game and discarded when the session ends.

6. Children's Privacy

CHORIDOR does not knowingly collect data from children under 13. If you believe a child has provided personal information, contact us and we will take appropriate action.

7. Changes

This policy may be updated at any time. The date at the top of this page reflects the most recent revision.
